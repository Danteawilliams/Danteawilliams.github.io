TIC-TAC-TOE

Game Objective:

Tic Tac Toe is a two-player game that is played on a 3 by 3 square grid. The first player to connect three "X"'s or "O"'s (vertically, horizontally or diagonally) is the winner.

Summary:

Tic Tac Toe is one of the most popular games, it fairly simple to learn to play and very competitive. This is a JavaScript application that uses the HTML & CSS on the GUI.
